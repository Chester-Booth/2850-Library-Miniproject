## Building & Running

To build or run the project, use one of the following tasks:

| Task            | Description      |
| -----------------|------------------ |
| `./amper test`  | Run tests for all modules    |
| `./amper build` | Build everything |
| `./amper run -m server`   | Run the server   |
| `./amper run -m init`   | Initialise the database   |
